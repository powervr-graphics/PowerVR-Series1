Do you have multiple Direct3D accelerator boards in your system ?
3D Control Center is a small application which allows you to easily use multiple Direct3D accelerator cards in one system at the same time. It places a "3D" icon the taskbar which quickly allows you to select which accelerator card you want to use for Direct3D acceleration. Native 3D applications are not affected by 3D Control Center - they will always run on the appropriate accelerator card.
The PowerVR driver packages, sample programs and documentation are provided as compressed ZIP archive files. If you do not have a utility to decompress ZIP archives, please download the latest version of WinZIP.
Some of the PowerVR developer documentation is also provided in Adobe Acrobat format. In order to read files in this format (.PDF) the Adobe Acrobat Reader is required. If you do not have the latest version, please download the latest version of Adobe Acrobat Reader.
